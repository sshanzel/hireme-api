# Install the dependencies

This is assuming you have `nvm` installed, if not, kindly install `nvm` on your machine.

Then run `nvm use` - the project contains `.nvmrc` so it will target the required node version.

## Getting Started

Depending on which one you used for installing the dependencies, you can run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
